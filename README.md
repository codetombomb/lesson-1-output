# Learning Output: puts and print

### Objectives
After this lesson, you will be able to:

- Write Ruby code that outputs text using `puts` and `print`
- Explain the difference between `puts` and `print`
- Predict how Ruby will format output before running the code


### Why Output Matters

Computers only do what they’re told — and that includes showing you things on the screen.

When you write a program, nothing happens unless you tell it to show something—just a blank screen.

In Ruby, we use `puts` and `print` to make the computer “talk back.”

This is how we see what our code is doing.

### Your First Output

Let’s make Ruby say something.

`puts "Hello, world!"`
